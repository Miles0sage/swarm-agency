# OpenClaw One-Pager 2026

## What is OpenClaw?

Multi-agent orchestration platform with sports betting + life automation capabilities. Thinks in workflows, not features. Optimizes cost/speed per task.

**Key Insight**: Only system generating revenue while executing tasks (sports arbitrage + prediction markets).

---

## Positioning

**For**: Founder-technologists seeking autonomous workflow orchestration
**Who**: Want to monetize their own agent outputs (sports picks, prediction bonds)
**Unlike**: Devin, Cursor, Manus (single-agent, cost-opaque)
**We**: Multi-agent + transparent costs + self-funding sports model
