# Passive Income Plan (3 Paths)

## Path 1: Sports +EV Betting (Fastest ROI)
- **Model**: XGBoost trained on 3 seasons NBA
- **Timeframe**: Immediate (games daily)
- **Typical edge**: 2-5% per bet
- **Monthly potential**: $2-5K from $10K bankroll
- **Tools**: sports_predict, sportsbook_odds, sportsbook_arb, bet_tracker
- **Status**: Model ready, tracking system live

## Path 2: Prediction Bonds (Passive)
- **Target**: High-probability contracts (>90% implied)
- **Markets**: Polymarket, Kalshi
- **Strategy**: Buy YES on locks, hold to maturity
- **Typical yield**: 3-8% APY
- **Monthly potential**: $500-2K from $100K capital
- **Tools**: polymarket_monitor (bonds scanner), polymarket_trade
- **Status**: Monitor tool ready, just need capital allocation

## Path 3: Picks-as-a-Service (Recurring)
- **Offer**: Daily +EV sports picks to subscribers
- **Price**: $9.99-29.99/month
- **Customer target**: Casual bettors, Discord/Telegram community
- **Tools**: sports_predict + analysis, bet_tracker for transparency
- **Monthly potential**: $5-10K from 500-1K subscribers
- **Status**: Model + tracking ready, need distribution channel
